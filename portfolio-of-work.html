SheCodes poject - Weather application

This is project was the final assignment from the SheCodes Plus workshop. The requirement was to build a weather application with the use of HTML, CSS, Javascript and API integration to an online weather service.

Along with building on the prior HTML and CSS knowledge that had been established, we worked extensively with vanilla JavaScript and learnt to incorpotate API integration to make this site fully functional. This course also offered an introduction to Bootstrap with the requirement use a grid layout in the project.
